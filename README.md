# Угадай число

Красивая и полностью рабочая игра «Угадай число от 1 до 100» на чистом JavaScript.

## Особенности
- Минимальное количество попыток — цель игры  
- Рекорд сохраняется в `localStorage` навсегда  
- Отдельный экран победы с анимацией  
- Два разных сообщения: новый рекорд / просто победа  
- Поддержка ввода по Enter  
- Адаптивный и современный дизайн (glassmorphism)  
- 100% vanilla JS — без библиотек

## Технологии
- HTML5
- CSS3 (Flexbox, стеклянные эффекты)
- Vanilla JavaScript
- localStorage

## Как запустить
1. Склонируй репозиторий
2. Открой `index.html` в браузере
3. Играй и бей рекорды!
